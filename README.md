# vue-elementui-
个人练手项目-vue+element简单使用

### 启动说明
``` bash
# 进入项目根目录
cd **

# 安装依赖文件
npm install

# 发布开发环境
npm run dev
```
#### 技术栈
vue + vue-router + webpack + elementui
